# Week 1: Linux Navigation

## Objective

Learn how to move around the Linux file system using the terminal.

## Commands

### pwd

Shows the current directory.

### ls

Lists files and folders.

### ls -la

Lists all files, including hidden files, with additional details.

### cd

Changes directories.

### mkdir

Creates a new directory.

### touch

Creates a new file.

### cp

Copies a file.

### mv

Moves or renames a file.

### rm

Deletes a file.
