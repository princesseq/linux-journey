# Linux Journey

## About This Repository

This repository documents my journey learning Linux using Ubuntu.

I created this project to build a strong foundation in Linux as I work toward becoming a Cloud Engineer. My goal is to develop practical skills through hands-on labs, problem-solving, and consistent practice rather than simply studying concepts.

Everything in this repository reflects what I have personally learned, practiced, and documented throughout my journey.

## Goals

* Learn Linux fundamentals
* Practice using the Linux terminal
* Understand file systems and permissions
* Learn networking basics
* Develop troubleshooting skills
* Build confidence using Linux in real-world scenarios

## Learning Approach

For each topic I study, I will:

1. Learn the concept.
2. Practice it on my Ubuntu laptop.
3. Document what I learned.
4. Record any challenges I faced.
5. Continue building on previous knowledge.

## Progress

This repository will continue to grow as I learn new Linux concepts and complete additional labs.
